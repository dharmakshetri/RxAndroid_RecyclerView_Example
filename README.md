# RxAndroid_RecyclerView_Example
RxAndroid Simple Display the data on RecyclerView

#Dependencies
```java
compile 'com.android.support:recyclerview-v7:25.0.1'
compile 'io.reactivex:rxandroid:1.2.1'
compile 'io.reactivex:rxjava:1.1.6'
```
